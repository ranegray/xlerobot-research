# Backlog Structure

Use these areas for issues, PRs, and Linear imports.

## Runtime

Launch files, fake hardware, real hardware bridges, ROS topic contracts, package build, and smoke tests.

## Harness

Episode state, guard events, abort paths, reset boundaries, intervention logging, and verifier output.

## Benchmark

Task protocols, repeated-run definitions, success metrics, reset protocol, and benchmark fixtures.

## Data

Rosbag policy, run cards, checksums, artifact links, calibration snapshots, and replication bundles.

## Policy

Learned-policy baselines, policy API contracts, inference constraints, and model artifact rules.

## Agent

Coding-agent boundaries, agent-facing command interfaces, contribution rules, generated-code permissions, and audit logs.

## Docs

Setup, launch, command, observation, recording, architecture, and reproduction documentation.
